## Recent Topics NG
Extension for phpBB - Adds a list with a number of recent topics to the board index.

Based on NV Recent Topics for phpBB 3.0, by Joas Schilling ([nickvergessen](https://github.com/nickvergessen))

#### Requirements
* phpBB 3.3.5 - 3.3.x
* PHP 7.1.3 - 8.3.x
